# Debug config params

## Compiler options

Set to compiler ```version 5```

## Simulator config

**CPU DLL:**

```
DARMSTL.DLL
```

**Parameter:**

```
-REMAP
```

**Dialog DLL:**

```
DARMSTL.DLL
```

**Parameter:**

```
-pSTM32F103RB
```

## ST-link Debugger

Debugger name: ```ST-link Debugger```

**Driver DLL:**

```
SARMCM3.DLL
```

**Parameter:**

None

**Dialog DLL:**

```
TARMSTL.DLL
```

**Parameter:**

```
-pSTM32F103RB
```

### Additionnal settings

ST-link Debugger -> Settings -> Flash and Download -> Check ```Reset and Run```
